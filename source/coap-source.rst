Kafka Connect CoAP Source
=========================

A Connector and Source to stream messages from a CoAP server and write them to a Kafka topic.

The Source supports:

1. DTLS secure clients.

Prerequisites
-------------

- Confluent 3.0.1
- Java 1.8
- Scala 2.11
